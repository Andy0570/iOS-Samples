# iOS 数据持久化方案


* NSUserDefaults （偏好设置）
* property list（plist，属性列表）
* NSKeyedArchiver NSKeyedUnarchiver（归档、解档）
* text file（Write 写入文件方式，永久保存在磁盘中）
* SQL databases（SQLite 数据库）
* Core Data
* KeyChain (钥匙串)
